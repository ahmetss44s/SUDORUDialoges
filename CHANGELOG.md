# Changelog

## [v1.0.5] — 2026-03-15

### ✨ Новый дизайн инвентаря торговца

- **Карточки предметов** — каждый товар отображается вертикальной карточкой:
  - Строка 1: название предмета (бумага с ценой в lore)
  - Строка 2: иконка предмета (сам предмет с полным описанием)
  - Строка 3: кнопка **Купить ×1** (зелёная)
  - Строка 4: кнопка **Купить ×5** (синяя, покупает 5 комплектов)
- **Панель описания** справа (колонки 6-8):
  - Иконка торговца
  - Описание торговца (разбито по строкам)
  - Информация об обновлении ассортимента

## [v1.0.4] — 2026-03-15

### 🐛 Исправления

- **TraderDialogMenu:** кнопка «Выход» теперь корректно закрывает диалог (`closeInventory` на main thread)
- **TraderShop:** при покупке игрок получает **чистый предмет** — без лора магазина (цены, "Нажми кнопку ниже" и т.д.)

## [v1.0.3] — 2026-03-15

### 🔧 Исправления

- **TraderDialogMenu:** удалён неиспользуемый импорт `DialogAction`
- **TraderDialogMenu:** добавлена аннотация `@SuppressWarnings("UnstableApiUsage")` — подавление предупреждений нестабильного Paper Dialog API

### ✨ Добавлено

- Файл `CHANGELOG.md` для отслеживания истории изменений

## [v1.0.2] — 2026-03-15 — Последний стабильный релиз

### 🔧 Исправления

- **Валюта:** исправлено снятие предметов-валюты из инвентаря игрока (`EventPriority.HIGHEST` + `updateInventory`)
- **HEX цвета:** исправлена обработка цветов `&#RRGGBB` в названиях (`Matcher.quoteReplacement` + `toUpperCase`)
- **TraderMenu:** исправлено отображение «предметов» в `/tradermenu` при открытии редактора (вкладка `/traderconfig items`)
- **NPE:** исправлен `NullPointerException` при обращении к пустым слотам/предметам

### ✨ Добавлено

- Автоматически создаются **3 встроенных торговца** с 5+ предметами при первом запуске плагина
- Новые предметы в пулах (слоты 5–6 видов на каждого торговца)
- Обновлён **README**: подробное описание команд, прав и настроек
